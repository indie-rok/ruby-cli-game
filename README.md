This is a FUN CLI game!

### Features

- Move around rooms
- Allow hearo to heal
- Allow hero to fight monsters
- Allow hero to win

### Instructions

1. Clone repo
2. ```
cd ruby-cli-game
```

3. ```
ruby main.rb
```

To test:
You should have installed rspec gem

```
gem install rspec
```

And then from home folder run

```
rspec
```
